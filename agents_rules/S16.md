### 16. Java 监听器接口（OnXxxListener）需要传**纯函数**，SAM 对象字面量在所有位置都失败
example in micro-habit\UTS_rules\SS2.md

**规则**：
- Java 风格的 `new X.OnXxxListener({...})` 在 UTS 中**不需要 `new`**——但仅此而已
- **场景 A**：`setOnXxxListener` 等实例方法 → **必须传纯函数** `function(args): T { ... }`
  - ❌ 错误：`setOnPreparedListener(MediaPlayer.OnPreparedListener({ onPrepared(mp) {} }))` → "实际类型为 'UTSJSONObject'，预期类型为 'Function1<MediaPlayer!, Unit>'"
  - ✅ 正确：`setOnPreparedListener(function(mp: MediaPlayer): void { ... })`
- **场景 B**：构造函数 + 监听器参数（如 `TextToSpeech(ctx, listener)`）→ **必须传纯函数** `function(args): T { ... }`
  - 错误示例：`TextToSpeech(ctx, TextToSpeech.OnInitListener({onInit}))` → "实际类型为 'UTSJSONObject'，预期类型为 'Function1<Int, Unit>'"
  - 正确：`TextToSpeech(ctx, function(status: number): void { ... })`
- 原因：UTS 的 SAM 接口（`OnXxxListener`）期望 Kotlin 函数类型 `Function1<T, R>`；对象字面量 `{onInit: ...}` 在**任何**位置（setter / 构造函数）都不触发 SAM 转换，被推断为 `UTSJSONObject`
- 适用于所有 `MediaPlayer.On*Listener`、`TextToSpeech.OnInitListener`、`View.OnClickListener` 等 SAM 接口
- 实际项目代码风格参考：`uni_modules/uts-floating-overlay/utssdk/app-android/FloatingOverlay.uts:188/204/223/287` 全部用纯函数

### 16. Java 监听器接口（OnXxxListener）需要传纯函数
```uts
// ❌ 全部编译失败：'Type mismatch: actual type is UTSJSONObject, but Function1<...> was expected'
const listener = new TextToSpeech.OnInitListener({ onInit(status: number) {} })
this.mp!!.setOnPreparedListener(new MediaPlayer.OnPreparedListener({ onPrepared(mp) {} }))
this.mp!!.setOnPreparedListener(MediaPlayer.OnPreparedListener({ onPrepared(mp) {} }))
this.mp!!.setOnCompletionListener(MediaPlayer.OnCompletionListener({ onCompletion(mp) {} }))
this.mp!!.setOnErrorListener(MediaPlayer.OnErrorListener({ onError(mp, what, extra) {} }))

// ✅ 正确：setOnXxxListener 传纯函数
this.mp!!.setOnPreparedListener(function(mp: MediaPlayer): void {
  mp.start()
})
this.mp!!.setOnCompletionListener(function(mp: MediaPlayer): void {
  cbs.onCompletion()
})
this.mp!!.setOnErrorListener(function(mp: MediaPlayer, what: number, extra: number): boolean {
  cbs.onError('error: ' + what + '/' + extra)
  return true
})

// ✅ 正确：构造函数 + 监听器 → 必须传纯函数
ttsInstance = TextToSpeech(ctx, function(status: number): void {
  if (status == TextToSpeech.SUCCESS) { ... }
})
```
